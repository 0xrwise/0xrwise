- ◈ **Android OS Exploitation** — This goes below root. I operate at the layer where manufacturer assumptions live and security models are drafted in pencil. Client-side cryptography, attack surface reduction, and adversarial threat modelling applied before a single line reaches production. The system survives even when the infrastructure does not.
